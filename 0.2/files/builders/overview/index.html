    Builders¶

    Builders use third-party services to source items to be added to the collection. Multiple builders can be used in the same collection from a variety of sources listed below.

    Builder

    Plex (Smart) builders create Smart Collections based on the metadata inside your Plex Server. Results are dynamic and will update as your library updates without the need to re-run Kometa.

    View Builder

    Popular Builders
    • Smart Label - Create a Smart Collection based on a specific label. The label can be generated by Kometa based on criteria.
    • Smart Filter - Create a Smart Collection based on the filter parameters provided.

    Builder

    Plex (Dumb) builders grab items based on the metadata inside your Plex Server. Results are static and require Kometa to re-run to update.

    View Builder

    Popular Builders
    • Plex All - Gets every movie/show in your library.
    • Plex Search - Gets every movie/show based on the search parameters provided.
    • Plex Collectionless - Gets every movie/show that is not in a collection.

    Builder

    TMDb builders grab items based on metadata and lists on TheMovieDb.org

    View Builder

    Popular Builders
    • TMDb Collection - Gets every movie/show in your library.
    • TMDb List - Gets every movie/show in your Watchlist.
    • TMDb Company - Gets the first episode of every show in your library.
    • TMDb Network - Gets the first episode of every show in your library.

    Builder

    Trakt builders grab items based on metadata and lists on Trakt.tv

    View Builder

    Popular Builders
    • Trakt List - Gets every movie/show in the Trakt List
    • Trakt Chart - Gets every movie/show in the Trakt Chart
    • Trakt Recommendations - Gets every movie/show in Trakt's Personal Recommendations for your User

    Builder

    MDBList builders grab items based on lists on MDBList.com

    View Builder

    Popular Builders
    • MDBList List - Gets every movie/show in a MDBList List.

    Builder

    IMDb builders grab items based on metadata and lists on IMDb.com

    View Builder

    Popular Builders
    • IMDb Chart - Gets every movie/show in an IMDb Chart like IMDb Top 250 Movies.
    • IMDb List - Gets every movie/show in an IMDb List.
    • IMDb Award - Gets every movie/show in an IMDb Event.
    • IMDb Search - Gets every movie/show in an IMDb Search. Can also be used for Keyword searches.

    Builder

    TVDb builders grab items based on metadata and lists on TheTVDb.com

    View Builder

    Popular Builders
    • TVDb Show - Grabs the specified series.
    • TVDb Movie - Gets the specified movies.
    • TVDb List - Gets every item in a TVDb List or TVDb UserList.

    Builder

    BoxOfficeMojo builders grab items based on metadata and lists on Boxofficemojo.com

    View Builder

    Popular Builders
    • Mojo World - Grabs items in the Worldwide Box Office chart.
    • Mojo Domestic - Grabs items in the Domestic Box Office chart.
    • Mojo Record - Grabs items in miscellaneous lists such as Weekend Records and Daily Records.

    Builder

    Tautulli builders grab items based on lists on Tautulli

    View Builder

    Popular Builders
    • Tautulli Popular - Gets every movie/show in the Tautulli Most Popular List
    • Tautulli Watched - Gets every movie/show in the Tautulli Most Watched List

    Builder

    Radarr builders grab items based on tags in Radarr

    View Builder

    Popular Builders
    • Radarr All - Gets every movie in Radarr
    • Radarr Taglist - Gets every movie with the specified tags

    Builder

    Sonarr builders grab items based on tags in Sonarr

    View Builder

    Popular Builders
    • Sonarr All - Gets every show in Sonarr
    • Sonarr Taglist - Gets every show with the specified tags

    Builder

    Letterboxd builders grab items based on lists on Letterboxd.com

    View Builder

    Popular Builders
    • Letterboxd List - Gets every movie in the Letterboxd list

    Builder

    ICheckMovies builders grab items based on lists on ICheckMovies.com

    View Builder

    Popular Builders
    • ICheckMovies List - Gets every show in the ICheckMovies list

    Builder

    AniDB builders grab items based on metadata and lists on AniDB.com

    View Builder

    Popular Builders
    • AniDB ID - Finds the anime specified by the AniDB ID
    • AniDB Popular - Finds every anime in AniDB's Popular Anime list
    • AniDB Tags - Finds every anime with the specified AniDB Tag

    Builder

    AniList builders grab items based on metadata and lists on AniList.com

    View Builder

    Popular Builders
    • AniList Top Rated - Gets every anime in AniList's Top Rated Anime list
    • AniList Popular - Gets every anime in AniList's Popular Anime list
    • AniList Trending - Gets every anime in AniList's Trending Anime list
    • AniList Search - Gets every anime specified by the AniList search parameters provided

    Builder

    MyAnimeList builders grab items based on metadata and lists on MyAnimeList.com

    View Builder

    Popular Builders
    • MyAnimeList Popular - Gets every anime in MyAnimeList's Most Popular Anime list
    • MyAnimeList Top All - Gets every anime in MyAnimeList's Top All Anime list
    • MyAnimeList Top Airing - Gets every anime in MyAnimeList's Top Airing Anime list
    • MyAnimeList Suggested - Gets every suggested anime in by MyAnimeList for the authorized user

    Builder

    Reciperr builders grab items based on lists on Reciperr.com

    View Builder

    Popular Builders
    • Reciperr List - Gets every movie returned by the Reciperr JSON data URL.

    Builder

    StevenLu builders grab items based on lists on StevenLu.com

    View Builder

    Popular Builders
    • StevenLu Popular - Gets every movie on StevenLu's Popular Movies List.
